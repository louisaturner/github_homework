In this project I learned how to display maps in different projections

To execute this, I used the QGIS software and changed the EPSG every time to create a new projection. For example, to visualize the Psuedo Mercator projection, I changed the EPSG to 3857 and then created a new print layout and then exported the image and pasted it on this webpage.
